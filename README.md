# Text Flow Chart

Text flow chart is creating flow charts from bulleted text.

## Usage

## Syntax

```
1. normal line
2. condition line
- if condition
  process in condition
  - nested if condition
    nested condition
- if another
  another process
  multi line
- if back to 1
  -> 1
3. repeat line
  3.1. indent lines without condition is considered "repeat"
  3.2. multi repeat
```
